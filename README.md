# R.I.M Herrería en General — Web

Sitio web de R.I.M Herrería en General (Mario Barrios).
Herrería, estructuras metálicas, portones, rejas y trabajos a medida.

## Estructura
- `index.html` — la web completa (un solo archivo, autocontenido)
- `hero_video.mp4` — video del hero

## Cómo verla localmente
Abrir `index.html` con doble clic en cualquier navegador.
Importante: `index.html` y `hero_video.mp4` deben estar en la misma carpeta.

## Cómo publicarla
### Vercel (recomendado)
1. Subir este repo a GitHub
2. En vercel.com → Add New → Project → importar este repo → Deploy

### GitHub Pages
1. Settings → Pages → Branch: main → Save
2. La web queda en https://USUARIO.github.io/rim-herreria/
